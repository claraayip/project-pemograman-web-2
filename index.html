Case Study

Income Calculator App

UI/UX Case Study

Problem

Banyak pengguna masih menghitung penghasilan secara manual yang kurang efisien dan rentan kesalahan.

Solution

Aplikasi yang membantu pengguna menghitung total penghasilan secara otomatis dengan antarmuka sederhana dan mudah digunakan.

Key Features

  • Input berbagai jenis penghasilan
  • Validasi input otomatis
  • Perhitungan total real-time
  • Tampilan data dalam tabel

Design Approach

Menggunakan pendekatan minimalis dengan fokus pada kemudahan penggunaan, visual hierarchy yang jelas, serta pengalaman pengguna yang intuitif.

Java Java Swing UI/UX Design
